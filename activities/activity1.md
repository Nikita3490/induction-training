# Activity 1: Launch Your First Two-Hour Session

## Session Focus
Choose one of the following focal problems and write it down before you begin:
- Designing a classroom activity that makes student thinking visible
- Aligning a lab task with a clear learning outcome and assessment strategy
- Creating a syllabus segment that balances concept introduction with practice

## Step 1: Self-Study
Select a short resource and capture the key insight:
- Resource:
- Key ideas:
- Questions raised:

Suggested self-study focus:
- How does constructive alignment connect learning outcomes, activities, and assessments?
- What student mental skills should this session make more visible and measurable?

## Step 2: Creation
Draft one instructional artifact using your chosen focus.
Possible artifacts:
- A 1-paragraph activity prompt for a 45-minute class
- A syllabus learning outcome with two aligned assessment criteria
- A lab design brief that states the goal, student task, and evidence of learning

### Creation checklist
- What is the target learning outcome?
- How will students show they are learning it?
- How does this task avoid mere recall and encourage reasoning or design?

## Framing Question
> What is the single student thinking skill this artifact should build, and how will I know whether it worked?
